
# meta_mind

In meta_mind, Two Wikipedia Featured Articles face
off in a fast-paced metadata duel. Choose the winner
in categories like pageviews, size, age, edit count
and more ...
Trust your instincts, make your pick, and then see how the
metadata decides!

Install by executing in a terminal 
``
git clone https://github.com/nesmomik/meta_mind.git
``

Change into the directory with 
``
cd meta_mind
``

Also please install the dependencies with
``
pip install -r requirements.txt 
``
or
``
uv add -r requirements.txt 
``

Enjoy!
